5) Para que serve a tag "label"? Para que serve o atributo "for" nessa tag?
Resposta: A tag "label" representa uma legenda para um item em uma interface de usuário. O atributo "for" faz um tipo de link entre a label e o input. Logo o "for" tem que apontar para o ID do input.

6) Na tag "option" para que serve o atributo "selected"?
Resposta: O SELECT tem, não só um identificador de abertura, mas também um identificador de encerramento. Dentro do SELECT podemos ter apenas uma sequência de identificadores OPTION.

7) Nas tags de formulário para que serve o atributo "required"?
Resposta: É usado para indicar que um determinando campo de formulário é obrigatório para o envio do mesmo.

8) Na tag form para que serve o atributo "action"?
Resposta: O atributo "action" define o local (através de uma URL) ao qual serão enviados todos os dados recolhidos do formulário.

9) Na tag form para que serve o atributo "method"?
Resposta: O atributo "method" define qual o método HTTP para enviar os dados. Ele pode ser "GET" ou "POST"

10) Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag "form"? Resposta: Deve-se colocar o atributo post, pois o método "post" é mais robusto e seguro do que "get", e "post" não possui limitações de tamanho
Qual valor deve ser informado no atributo "method" na tag "form" em um formulário que será usado para fazer upload de arquivos? Resposta: Deve-se colocar o atributo get, este método anexa os dados do formulário à URL em pares nome/valor.
