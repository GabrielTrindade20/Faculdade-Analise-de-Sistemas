1) Qual a finalidade dos atributos method e action da tag form
Method especifica como enviar os dados e o action identifica a página que os dados serão enviados

2) Quais são os valores possíveis para o atributo method
get ou post

3) Qual é a tag que deve ser usada para criar uma caixa de texto de uma linha?
a tag input com o type="text"

4) Como fazer para que em um grupo de quatro botões de rádio, apenas um deles fique ativo de cada vez? Qual é o valor de type usado para criar um botão de rádio?
Para que botões fiquem ativos um por vez, o "name" tem que ser o mesmo e pode possuir valores diferentes

5) Como conectar a tag label com um elemento de um formulário?
Crie o formulário dentro da label

6) Para que servem os elementos fieldset e legend?
Os 2 servem para agrupar e organizar elementos, o fieldset é usado para agrupar conjuntos de campo e a legend é usado para dar uma descrição de um conteúdo.

7) Para que serve o elemento input quando o type é igual a submit?
Serve para enviar os dados que foram digitados pelo usuário

8) Para que serve o elemento input quando o type é igual a reset?
Serve para restaurar todos os elementos de entrada do formulário, aos seus valores iniciais

9) Como criar uma caixa de texto que tenha mais de uma linha?
